# 🎬 Cinephix

A sleek, full-stack movie recommendation engine that uses **Gemini 2.0 Flash** to suggest films based on your current mood and personality.



## 🌟 Features
* **Personality-Based Logic**: Uses LLM reasoning to match films to complex human emotions.
* **Responsive Design**: A modern, dark-themed UI built with HTML5 and CSS3.
* **Flask Backend**: A robust Python server handling API communication and data parsing.
* **Real-time Interaction**: Fetching recommendations without page reloads using JavaScript Fetch API.

## 🛠️ Tech Stack
* **Frontend**: HTML5, CSS3 (Glassmorphism), Vanilla JavaScript.
* **Backend**: Python, Flask.
* **AI Engine**: Google Gemini API (google-genai).
* **Environment Management**: Python-dotenv.

## 🚀 Getting Started

### Prerequisites
* Python 3.10+
* A Google AI Studio API Key
